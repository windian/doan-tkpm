# CHÚ Ý
SpeechRecognition chỉ được hỗ trợ bởi các trình duyệt Chorme, Safari.

Firefox chỉ hỗ trợ SpeechRecognition trong bảng Firefox Nightly.

# Cách install và start
## Cài đặt
cd frontend
npm install
cd ../backend
npm install

## Start frontend và backend
### Mở command line từ root
cd frontend
ng serve

### Mở một command line khác từ root
cd backend
npm run serve
